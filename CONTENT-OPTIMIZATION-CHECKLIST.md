### Content Structure

#### Introduction (First 100 words)
- [ ] Hook reader with compelling opening
- [ ] Include primary keyword in first paragraph
- [ ] State the problem/question
- [ ] Promise the solution
- [ ] Add table of contents for long posts (1500+ words)

#### Main Content
- [ ] Use H2 for main sections (3-5 sections)
- [ ] Use H3 for subsections (2-3 per H2)
- [ ] Include primary keyword in at least one H2
- [ ] Use LSI keywords in other headings
- [ ] Keep paragraphs short (2-4 sentences)
- [ ] Use bullet points and numbered lists
- [ ] Add examples and case studies
- [ ] Include statistics and data (with sources)

#### Conclusion
- [ ] Summarize key points
- [ ] Include call-to-action
- [ ] Link to related content
- [ ] Encourage comments/engagement

### Content Quality

#### Word Count
- [ ] Minimum 1,500 words for competitive keywords
- [ ] 2,500+ words for highly competitive keywords
- [ ] 3,000+ words for pillar content
- [ ] Match or exceed top 3 competitors

#### Readability
- [ ] Use simple language (8th-grade reading level)
- [ ] Short sentences (15-20 words average)
- [ ] Short paragraphs (2-4 sentences)
- [ ] Use transition words
- [ ] Avoid jargon (or explain it)
- [ ] Use active voice
- [ ] Target Flesch Reading Ease: 60-70

#### Keyword Optimization
- [ ] Primary keyword density: 1-2%
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in at least one H2
- [ ] Primary keyword in conclusion
- [ ] LSI keywords throughout content
- [ ] Natural keyword placement (no stuffing)
- [ ] Keyword variations and synonyms
